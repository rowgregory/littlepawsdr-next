# Contributing

This project does **not** accept external contributions.

Little Paws Dachshund Rescue is proprietary software owned and maintained solely
by [Sqysh](https://sqysh.com). The repository is public for reference and
transparency only.

- **Pull requests will not be reviewed or merged.** Any PR opened against this
  repository will be closed without review.
- **Issues are not monitored for feature requests or bug reports** from outside
  parties.
- **The code may not be reused.** See [LICENSE](./LICENSE) for terms.

If you have a licensing question or a business inquiry, please contact Sqysh
Studio directly at https://sqysh.com.

Thank you for understanding.
